# Capgemini-desafio-de-programacao
Esta é o repositório de resolução de exercícios propostos em uma das fases do processo seletivo da [Academia Técnica Capgemini 2022!](https://capgemini.proway.com.br/).

## Tech Stack
* Javascript puro
  
## Dependências 
 * Algum editor de código, de preferência o [VSCode](https://code.visualstudio.com/).
 * [Node.js](https://nodejs.org/en/) para executar o código Javascript.
 
## Como iniciar
Abra o terminal na pasta onde estão os arquivos .js e insira o código abaixo:
~~~
node Questaox.js
~~~
Obs: "x" é o número da questão/arquivo que você queira executar
